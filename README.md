##   Overview
This project is a Machine Learning based Heart Disease Prediction System developed using Python. 
The system predicts whether a person is likely to have heart disease based on medical parameters.
##  Technologies Used
-Python
-Pandas
-NumPy
-Scikit-learn
-Streamlit
##  Features
-User-friendly interface
-Predicts heart disease risk
-Machine Learning model integration
-Fast and simple prediction system

-scalar.pkl
-heart.csv
